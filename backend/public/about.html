About the Project

Built with Node.js,
made to last.

This project demonstrates a full-stack web application using Node.js and Express on the backend, MongoDB for data storage, and clean HTML pages on the frontend — all communicating through a RESTful API.

⚡
Express Backend
Lightweight REST API with proper error handling, CORS support, and JSON responses.
🗄️
MongoDB / Mongoose
Schema-based data modeling with Mongoose for clean, validated database operations.
🌐
HTML Frontend
Vanilla HTML pages with API integration — no framework overhead, just clean code.
🔒
User Management
Signup and user listing endpoints, ready to extend with authentication.
